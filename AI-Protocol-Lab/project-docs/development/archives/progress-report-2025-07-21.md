## 📊 项目概览

### 🎯 项目目标
开发一个现代化的加密货币市场智能平台，提供实时数据、新闻资讯、交易功能和账户管理。

### 🏗️ 技术栈
- **前端**: React 18 + Vite
- **UI框架**: Material-UI (MUI)
- **图表库**: Recharts
- **状态管理**: React Hooks
- **路由**: React Router
- **图标**: Cryptocurrency Icons + Material Icons
- **样式**: CSS-in-JS (styled-components)
