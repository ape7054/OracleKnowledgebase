## 如何使用

1. **更新联系方式**：编辑 `src/app/[locale]/about/page.tsx`，将联系方式链接替换为你的真实信息
2. **自定义内容**：根据需要修改 `src/messages/zh.json` 和 `src/messages/en.json` 中的 `about` 部分
3. **添加照片**：可以将头像 emoji 替换为真实照片
4. **扩展内容**：根据个人需要添加更多章节或信息
